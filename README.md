## 🚀 部署到 GitHub Pages

```bash
git init
git add .
git commit -m "init: 紫微斗數AI大師 PWA"
git branch -M main
git remote add origin https://github.com/<username>/<repo>.git
git push -u origin main
```

然後在 GitHub repo → **Settings → Pages** → Source 選 `main` 分支，等待幾分鐘後即可訪問。

## 📱 安裝 PWA
